  • Problem: Count the number of rooted planar n-ary trees with a given number of leaves. This generalizes the n=2 case, which yields the Catalan numbers.
  • Method: Use generating functions. The set of n-ary trees T is defined by the recursive equation T ≅ x + T^n, where x represents a leaf.
  • Solution: To solve for T as a power series in x, the equation is rewritten as x = T^n - T. This requires finding the compositional inverse of the formal power series f(z) = z^n - z.
  • Tool: The Lagrange inversion theorem is used to find the coefficients of the inverse power series. The theorem utilizes Bell polynomials, which are combinatorial objects that count set partitions.
  • Result: The number of rooted planar n-ary trees with (n-1)k + 1 leaves is given by the formula: 
    (1 / ((n-1)k + 1)) * (nk choose k)
  • Verification: For n=2, this formula correctly simplifies to the Catalan numbers.
