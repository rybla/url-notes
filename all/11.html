New paradigm for psychology just dropped
psychologycyberneticsphilosophy of sciencecognitive sciencecontrol theorymental modelsscientific paradigmpsychopathologyemotionspersonalityresearch methodologyfalsifiability

  • Critique of Modern Psychology:

    • Lacks a scientific paradigm, defined as a system of fundamental units and rules.
    • Current research is often "naive" (atheoretical data collection) or "impressionistic" (studying ill-defined, tautological abstractions like "zest for life").
    • This leads to non-cumulative, non-falsifiable findings and a failure to explain underlying mechanisms.

  • Proposed Paradigm: Cybernetic Psychology:

    • Core Unit: The control system (e.g., a thermostat), which seeks to minimize the "error" between a current state and a target state.
    • Core Rule: The mind is a stack of competing control systems, each regulating a specific variable (e.g., nutrition, sociality, body temperature).
    • Emotions: Defined as error signals from a control system (e.g., hunger, loneliness, pain).
    • Happiness: The metabolic-like reward for correcting an error, not an emotion itself.

  • Implications of the Paradigm:

    • Personality: Arises from individual differences in the set points and sensitivities of various control systems.
    • Psychopathology: Results from specific malfunctions within a control system's components (e.g., a broken error-feedback loop causing depression-like symptoms). This provides a mechanistic alternative to symptom-based classification.
    • Research Methodology: Shifts focus to discovering and isolating control systems through methods like prevention, knockout, and behavioral exhaustion.

  • Value Proposition:

    • The paradigm's primary strength is not its guaranteed correctness, but its overturnability.
    • By providing a strong, specific, and mechanistic model (like behaviorism was), it offers a concrete framework that can be rigorously tested, falsified, and built upon, fostering genuine scientific progress.
