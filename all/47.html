  • Untyped λ-Calculus: A model of computation based on variable substitution where terms are functions that act on other terms.
  • λ-Term Syntax: Inductively defined with three forms:
    • Variable: x
    • Application: ap(M₁; M₂) (conventional notation: M₁ M₂)
    • Abstraction: λ(x.M) (conventional notation: λx.M)
  • Structural Induction: The inductive definition of λ-terms enables proofs by rule induction (structural induction) on properties like term size and depth.
  • Binding: The λ operator binds variables. In λx.M, x is bound within M.
  • Free Variables (FV): A variable is free if it is not bound by a λ abstraction. The judgment x ∈ FV(M) is defined inductively:
    • x ∈ FV(x)
    • x ∈ FV(ap(M₁; M₂)) if x ∈ FV(M₁) or x ∈ FV(M₂)
    • x ∈ FV(λ(y.M)) if x ≠ y and x ∈ FV(M)
